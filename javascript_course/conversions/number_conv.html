Number Conversion

Numeric conversion occurs automatically when mathematical operations are applied to non-numeric values. This means that JavaScript tries to convert the values into numbers where necessary.

Example

alert("10" / "2"); // 5, strings are converted to numbers

You can also explicitly convert a value into a number using the Number() function:

Example

let userInput = "456";
alert(typeof userInput); // string

let numericInput = Number(userInput);
alert(numericInput); // 456
alert(typeof numericInput); // number

This is especially useful when values come from user input forms or text sources. For example, form fields return string values even if the user enters numbers.

Invalid Conversions

If a string doesn't represent a valid number, the conversion result will be NaN (Not a Number): 

let ageInput = Number("some random text");

alert(ageInput); // NaN
alert(typeof ageInput); // number

Numeric Conversion Rules

Value Converts to
undefined NaN
null 0
true 1
false 0 
"  789  "
789
"99xyz" NaN

More Examples

alert(Number("   789   "));     // 789
alert(Number("abc123"));        // NaN
alert(Number(true));            // 1
alert(Number(false));           // 0
alert(Number(undefined));       // NaN
alert(Number(null));            // 0

Boolean Conversion

Boolean conversion is one of the simplest types of data conversions in JavaScript.

You can convert any value to a boolean using the Boolean(value) function. Let's look at some examples: 

alert( Boolean(10) );          // true
alert( Boolean(0) );           // false

alert( Boolean("text") );      // true
alert( Boolean("") );          // false

alert( Boolean(null) );        // false
alert( Boolean(undefined) );   // false

alert( Boolean(NaN) );         // false
alert( Boolean("false") );     // true - it's a non-empty string

Important Notes

  • All non-zero numbers (positive or negative) are considered true.
  • All non-empty strings are also considered true, including strings like "0" and " " (a space).
  • The following values are the only ones considered false in JavaScript: 
    • 0
    • "" (empty string)
    • null
    • undefined
    • NaN

Examples

let result = Boolean("JavaScript");
alert(result); // true

let emptyValue = "";
alert(Boolean(emptyValue)); // false

let valueWithSpace = " ";
alert(Boolean(valueWithSpace)); // true

let isZero = Boolean(0);
alert(isZero); // false

let isNumber = Boolean(-25);
alert(isNumber); // true

Boolean conversion is especially useful when validating input, toggling flags, or checking conditions based on user data.

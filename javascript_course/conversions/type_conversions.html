Type Conversions

In JavaScript, values are often automatically converted from one type to another when used with different operators or functions. This behavior is known as type coercion.

Example:

Some functions like alert() convert any value into a string before displaying it: 

let temperature = 28;
alert(`Today's temperature is ${temperature}`); // "28" is converted to a string

Mathematical operations automatically convert values to numbers:

let width = "40";
let height = "5";
let area = width * height; // "40" and "5" are converted to numbers
alert(area); // Outputs: 200

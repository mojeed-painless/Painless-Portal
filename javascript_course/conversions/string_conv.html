String Conversion

String conversion occurs when a value needs to be represented as text. This happens automatically in certain situations as previously discussed.

We can also manually convert any value into a string using the String() function. 

let existingUser = false;
alert(typeof existingUser); // boolean

existingUser = String(existingUser); // now it's a string
alert(typeof existingUser); // string

As shown above, String() changes the data type of false to the string "false".

Other Examples

// Number to string
let score = 100;
let scoreText = String(score); 
alert(scoreText);   // "100"

// null to string
let response = null;
let responseText = String(response); 
alert(responseText)     // "null"

// undefined to string
let value;
let valueText = String(value); 
alert(valueText)    // "undefined"

Why Use String Conversion?

  • To display values on the screen using alert(), document.write(), or console.log()
  • To store data in text format (e.g. sending data over a network)

Basic Operators

In JavaScript, we use various operators to perform operations on values (also known as operands). Some operators you're already familiar with from basic math, such as:

  • + (addition),
  • - (subtraction),
  • * (multiplication),
  • / (division), and so on.

However, in programming, we often need to understand how many operands an operator works with. This gives us two types of operators:

Terms: Unary, Binary, and Operand

You need to understand these three basic terminologies to give you edge in understanding how operators actually work

Operand

An operand is the value the operator works on.

For example, in the expression 8 * 4, the operands (also called arguments) are 8 and 4.

The operator * performs multiplication on those two operands.

Unary Operators

A unary operator works with only one operand.

Example: the negation operator (-):

let a = 7;
a = -a;
alert(a); // -7

In the code above:

  • We had a value 7 assigned to a.
  • Then, a = -a; applied unary negation, reversing the sign.
  • As a result, a became -7.

Binary Operators

A binary operator works with two operands.

Example: subtraction (-):

let m = 10, n = 4;
alert(m - n); // 6

Here:

  • The binary operator - subtracts the right operand n (which is 4) from the left operand m (which is 10), resulting in 6.

Even though the same symbol - is used in both cases above (unary and binary), they are considered two different operators:

  • The unary - reverses the sign.
  • The binary - subtracts one value from another.

String Concatenation

In JavaScript, the binary plus operator (+) typically adds numbers. However, when it's used with strings, it performs concatenation â€” that is, it joins the strings together into one.

Example:

let greeting = "hello" + "world";
alert(greeting); // helloworld

When using +, if at least one operand is a string, JavaScript converts the other operand to a string and then concatenates.

Example: Mixing strings and numbers

alert('5' + 3); // "53"
alert(7 + '2'); // "72"

In both cases above, the number is converted into a string and then joined.

Operator Order Matters

Operators in JavaScript are evaluated from left to right. This affects the final result when combining strings and numbers.

Example: First numbers, then string

alert(10 + 10 + '5'); // "205"

  • '5' is a string â†’ 20 + '5' becomes "205"

Example: Starting with a string

  • '5' + 10 becomes "510"
  • "510" + 10 becomes "51010"

Once a string is introduced, all subsequent additions are treated as string concatenations.

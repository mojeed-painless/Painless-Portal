Maths Operators

JavaScript supports several basic mathematical operations, including:

  • Addition +
  • Subtraction -
  • Multiplication *
  • Division /
  • Remainder %
  • Exponentiation **

These operators can be used with numbers to perform calculations. We have no issue with the first four operators as they are straightforward. However, the last two operators, % and **, may require some explanation. Let's explore each of them in detail:

Remainder

The remainder operator % returns the remainder left over when one number is divided by another. It uses the percentages sign but it is in no way related to percentages.

Examples:

let x = 11;
let y = 4;
alert( x % y ); // 3, because 11 divided by 4 leaves a remainder of 3

let a = 15;
let b = 6;
alert( a % b ); // 3, remainder of 15 divided by 6

let p = 20;
let q = 5;
alert( p % q ); // 0, because 20 is evenly divisible by 5

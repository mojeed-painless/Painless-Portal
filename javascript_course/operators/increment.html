Increment & decrement

In JavaScript, one of the most common operations is increasing or decreasing a number by 1. JavaScript provides special shorthand operators for this:

  • Increment (++) increases a variable's value by 1.
  • Decrement (--) decreases a variable's value by 1.

Basic Usage

let score = 10;
score++;  // same as score = score + 1
alert(score); // 11

let lives = 5;
lives--;  // same as lives = lives - 1
alert(lives); // 4

Prefix vs Postfix

The ++ or -- can appear before or after a variable:

  • Prefix: ++value or --value
  • Postfix: value++ or value--

Both increment or decrement the value, but they behave differently when used in expressions.

The difference is visible when the result of the operation is used immediately.

Prefix Example

let points = 3;
let result = ++points;  // Increments first, then returns the new value

alert(result); // 4

Postfix Example

let energy = 3;
let result = energy++;  // Returns the current value, before it tries to increment

alert(result); // 3

Summary

If you only need to update the value, either form works:

let level = 0;

level++;
++level;

alert(level); // 2

But if you want to use the result of the operation immediately, choose wisely:

  • ++var: returns new value
  • var++: returns old value

Increment/Decrement in Expressions

These operators can also be used inside mathematical expressions.

let multiplier = 2;
let value = 1;

alert(5 * ++value); // 10 (value becomes 2 before the multiplication)

let step = 1;

alert(5 * step++); // 5 (uses 1 first, then step becomes 2)

However, this can reduce code readability. A cleaner style is to keep expressions simple: 

let multiplier = 2;
let value = 1;

++value

alert(5 * value); // 10

let steps = 1;

alert(5 * steps);  // 5
steps++;

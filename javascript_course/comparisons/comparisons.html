Comparisons

In mathematics, comparison operators are used to evaluate the relationship between two values. JavaScript uses similar comparison logic but with specific syntax.

Available Comparison Operators

  • Greater than: x > y
  • Less than: x < y
  • Greater than or equal to: x >= y
  • Less than or equal to: x <= y
  • Equality check: x == y (double equals checks value equality)
  • Inequality check: x != y

Results of Comparison

Every comparison operation returns a boolean value:

  • true â€“ if the comparison is correct
  • false â€“ if the comparison is incorrect

Examples:

alert(7 < 10);   // true
alert(4 == 5);   // false
alert(12 != 6);  // true

Using Comparison in Assignments

You can assign the result of a comparison to a variable. This allows you to reuse the boolean result in logic or conditions. 

let isOlder = 21 > 18;

alert(isOlder); // true

Important Note

Always remember: = is for assignment, while == is for comparison. Using them interchangeably can lead to bugs.

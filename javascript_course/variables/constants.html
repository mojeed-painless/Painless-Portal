Constants

What is a Constant?

In JavaScript, a constant is a variable whose value is fixed , and cannot be changed once assigned. Constants are declared using the const keyword and must be initialized(assigned) at the time of declaration.

Declaring a Constant

To declare a constant, use the const keyword instead of let or var:

const myBirthday = '28.11.1990';

Attempting to change the value of a constant will throw an error:

const myBirthday = '28.11.1990';

myBirthday = '11.08.2000'; // Error: Assignment to constant variable!

Why Use Constants?

  • They prevent accidental value changes.
  • They make your code more predictable and easier to debug.
  • They help communicate that certain values should remain unchanged.

When a programmer is confident that a value will never change, it's best to declare it with const to enforce that immutability and make the intention clear to anyone reading the code.

Uppercase Constants

When using constants to store values that are fixed and known before runtime (like color codes, tax rates, or configuration values), it is common practice to write them in uppercase with underscores: 

const COLOR_RED = "#F00";
const COLOR_GREEN = "#0F0";
const COLOR_BLUE = "#00F";
const COLOR_ORANGE = "#FF7F00";

let color = COLOR_ORANGE;
alert(color); // Outputs: #FF7F00

When to Use Uppercase

Use uppercase naming only for values that:

  • Are hard-coded and known before execution.
  • Serve as meaningful aliases for raw values.

In summary, use const for any variable whose value should not change. Use uppercase naming only when you're defining a fixed, hard-coded value.

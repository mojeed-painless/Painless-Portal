Javascript Code Structure

Every programming language has building blocks, and in JavaScript, it all starts with statements, semicolons, and comments. Mastering these early gives you a strong foundation.

What are Statements?

A statement is a command or instruction that the browser executes.

Example:

alert('Hello, world!')

This is a statement that tells the browser to show a pop-up message. JavaScript code is made up of many such statements.

You can write multiple statements in one line using a semicolon ;: 

alert('Hey'); alert('painless')

But for better readability, it's common to write each statement on a new line:

alert('Hey');
alert('painless');

Semicolons

In JavaScript, semicolons are used to separate statements. While they can often be omitted when using line breaks, it's considered best practice to include them.

This works fine:

alert('Hey')
alert('painless')

JavaScript automatically inserts semicolons where it thinks they're needed â€” a feature called Automatic Semicolon Insertion (ASI). But ASI isn't perfect. For example:

["apple", "pawpaw"].forEach(alert);

This will cause an error. Why? Because JavaScript reads it like this:

alert("Hello")["apple", "pawpaw"].forEach(alert);

Always add semicolons between statements especially as a beginner!. It's safer and more readable.

Data types 1

A value in JavaScript is always of a specific type. For example, it could be a number, a string, or something else.

There are eight basic data types in JavaScript:

  • Number
  • BigInt
  • String
  • Boolean
  • Null
  • Undefined
  • Object
  • Symbol

It is important to note that data types are divided into two main categories: Primitive and Non-Primitive types.

Primitive Data Types

Primitive types are the most basic data types. They represent a single value and are immutable (cannot be changed once created).

Seven of the eight data types are primitive:
Number, BigInt, String, Boolean, Null, Undefined, Symbol,

Non-Primitive Data Types

Non-primitive types (also known as reference types) are more complex. They can store multiple values and are mutable.

Only the Object is non-primitive

Note: Arrays and functions are technically objects in JavaScript, so they fall under the non-primitive category.

We can assign any type of value to a variable. A variable can first store a value of one type and later be reassigned to a value of a different type. For example: 

let info = "Welcome!"; // stores a string
info = 2025; // now stores a number
// no error

Programming languages like JavaScript that allow this flexibility are called dynamically typed. This means that while data types exist, variables are not permanently tied to any one type.

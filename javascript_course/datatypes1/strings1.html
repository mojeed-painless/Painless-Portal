Strings

In JavaScript, a string must be surrounded by quotes.

let city = "Lagos";
let city2 = 'Abuja is great too';
let message = `Welcome to ${city}`;

There are 3 types of quotes used for Javascript strings:

  • Double quotes: "Hello"
  • Single quotes: 'Hello'
  • Backticks: `Hello`

Double and single quotes are considered ‚Äúsimple‚Äù quotes and behave the same way in JavaScript.

Backticks

Backticks, however, have extended functionality. They allow us to embed variables and expressions inside a string using ${‚Ä¶}. For example: 

let user = "Tosin";

// embed a variable
alert(`Hi, ${user}!`); // Hi, Tosin!

// embed an expression
alert(`2 multiplied by 5 is ${2 * 5}`); // 2 multiplied by 5 is 10

The expression inside ${‚Ä¶} is evaluated and the result is inserted into the string. We can embed variables, arithmetic expressions, or more complex logic inside it.

Important: Only backticks support this kind of embedding. If we use other types of quotes, the expression will not be evaluated: 

alert("2 multiplied by 5 is ${2 * 5}"); // 2 multiplied by 5 is ${2 * 5}

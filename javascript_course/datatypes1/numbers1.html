Numbers

The number type in JavaScript represents both integers and floating-point (decimal) numbers. 

let price = 250; // integer
price = 89.75; // floating-point

JavaScript supports several arithmetic operations such as: multiplication (*), division (/), addition (+), subtraction (-), and more.

In addition to regular numbers, JavaScript has some special numeric values that also belong to the number type: Infinity, -Infinity, and NaN.

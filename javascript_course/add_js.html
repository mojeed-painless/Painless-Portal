Link JavaScript to HTML

To use JavaScript in your web pages, you need to insert it inside the HTML file using the <script> tag.

There are three main ways to add JavaScript to your HTML:

  1. Inline JavaScript – JavaScript code added directly to an HTML element’s attribute.
  2. Internal JavaScript – Code written inside a <script> tag within the HTML file.
  3. External JavaScript – Code placed in a separate file and linked to the HTML file.

Inline JavaScript

You can add JavaScript code directly to HTML elements using the onclick or other event attributes: 

<button onclick="alert('Hello!')"> Click me </button>

Internal JavaScript

Write your JavaScript inside a <script> tag in the HTML file:

<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>

    <h1>Welcome</h1>

    <script>
        alert("Hello from JavaScript!");
    </script>

</body>
</html>

External JavaScript

You can also write JavaScript in a separate file (e.g., script.js) and link it to your HTML: 

<!DOCTYPE html>
<html>
<head>
    <title>External JS</title>
    <script src="script.js"></script>
</head>
<body>
    <h1>External JavaScript Example</h1>
</body>
</html>

Then in script.js:

console.log("This message comes from an external JS file!");

Best Practice

  • Use external files for better code organization and reuse.
  • Place <script> tags just before the closing </body> tag to make sure the page loads first.

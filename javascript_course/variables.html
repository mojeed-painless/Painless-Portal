What is a Variable?

A variable is a container used to store data. This data can be text, numbers, or other types of information.

In JavaScript, you create (or declare) a variable using the let keyword.

Variable Declaration

Declaration means introducing a variable to the program so JavaScript knows it exists and sets aside space in memory for it. 

let message;

This tells the browser: "Prepare a container called message â€” Iâ€™ll put something inside it soon."

Variable Assignment

Assignment Assignment means giving a value to a declared variable using the equals sign =. 

message = 'Hello';

This stores the string 'Hello' inside the variable message.

You can access it any time:

alert(message); // shows "Hello"

Declare and Assign in One Step

Itâ€™s common to create a variable and store a value in it at the same time:

alert(message); // Hello!

Declaring Multiple Variables

You can also declare multiple variables in one go:

let user = 'John', age = 25, message = 'Hello';

But for better clarity, itâ€™s usually better to put each one on its own line:

                   
let user = 'John';

let message = 'Hello';

Updating a Variable

You can change what's inside a variable at any time:

let message = 'Hello'; 
message = 'Welcome';
alert(message); // Welcome!

The old value is replaced by the new one.

Copying a Variable

You can copy the content of one variable into another:

let greeting = 'Good morning!'; 
let message;

message = greeting;

alert(greeting); // Good morning!
alert(message); // Good morning!

Both variables now hold the same value, though they're still separate lockers.

Declaring the Same Variable Twice

Once you've declared a variable with let, you should not declare it again in the same scope. Trying to do so will result in an error: 

let message = "This";

// This will throw an error
let message = "That"; // SyntaxError: 'message' has already been declared

If you want to update the value, just reassign it:

message = "That";// Correct way

A Note on var (The Old Way)

In older JavaScript code, you might see var instead of let:

var message = 'Hello';

It works similarly to let but behaves differently in certain technical ways. For now, just know that let is the modern and preferred way to declare variables. We'll cover the differences later.

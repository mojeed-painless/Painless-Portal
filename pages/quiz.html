🧪 Quiz: Test Your Understanding

  1. Why was CSS introduced?
    A. To replace HTML
    B. To add animations
    C. To simplify styling and layout
    D. To store images

  2. Which tag was originally used in HTML 3.2 to control font appearance?
    A. <style>
    B. <color>
    C. <font>
    D. <text>

  3. What is the benefit of using an external CSS file?
    A. Faster HTML loading
    B. You can write JavaScript in it
    C. Centralized style management
    D. HTML becomes invisible

✅ Answers:

  • 1 → C. To simplify styling and layout
  • 2 → C. <font>
  • 3 → C. Centralized style management

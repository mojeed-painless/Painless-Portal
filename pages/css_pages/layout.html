Layout

Display Property

Block-level elements: Start on a new line and take full width. Examples include <div>, <h1> to <h6>, <p>, <section>

Inline elements: Appear on the same line and take only the required width. Examples include <span>, <a>, <img>, <strong>

You can override these using the display property.

display: block; / display: inline;

<span class="block-example">span as block</span>
<div class="inline-example">div as inline</div>

.block-example {
  display: block;
  background-color: lightblue;
}

.inline-example {
  display: inline;
  background-color: lightcoral;
}

display: none; vs visibility: hidden;

display: none: Hides the element and removes it from the flow.

visibility: hidden: Hides the element, but it still takes up space.

<p class="none-example">This won't be visible at all.</p>
<p class="hidden-example">This is hidden but still takes space.</p>

.none-example {
  display: none;
}

.hidden-example {
  visibility: hidden;
}

width and margin: auto;

width restricts the full width of a block element.

margin: auto centers the element horizontally.

<div class="center-box">Centered Box</div>

.center-box {
  width: 300px;
  margin: auto;
  background-color: lightgreen;
}

overflow Property

Controls how content is handled when it overflows its container.

  • visible – default, overflow is shown.
  • hidden – overflow is clipped.
  • scroll – always adds scrollbars.
  • auto – adds scrollbars when necessary.
<div class="scroll-box">
  This is a long content that overflows: Lorem ipsum dolor sit amet consectetur, adipisicing elit. Nemo voluptatem, necessitatibus sapiente eligendi sint quaerat nisi neque ipsa quos. Quaerat porro sapiente dolor corrupti cupiditate exercitationem voluptatibus soluta excepturi sint!
</div>

.scroll-box {
  width: 200px;
  height: 60px;
  overflow: auto;
  background-color: #eee;
}

Position Property

  • static: Default.
  • relative: Positioned relative to its normal spot.
  • absolute: Positioned relative to the nearest positioned ancestor.
  • fixed: Fixed to the screen.
  • sticky: Scrolls with the page and then "sticks".
<div class="relative-box">Relative Box</div>
<div class="absolute-box">Absolute Box</div>

.relative-box {
  position: relative;
  top: 20px;
  left: 40px;
  background-color: orange;
}

.absolute-box {
  position: absolute;
  top: 10px;
  left: 100px;
  background-color: teal;
}

z-index Property

Defines the stack order of elements (higher z-index = on top).

<div class="z-box z1">Box 1</div>
<div class="z-box z2">Box 2 (on top)</div>

.z1 {
  background-color: red;
  z-index: 1;
}

.z2 {
  background-color: blue;
  z-index: 2;
}

Float and Clear Property

float: Makes elements float beside each other.

clear: Stops the float behavior.

<div class="float-left">Left Float</div>
<div class="float-right">Right Float</div>
<div class="clear-fix">This clears the floats above.</div>

.float-left {
  float: left;
  width: 150px;
  height: 100px;
  background-color: pink;
}

.float-right {
  float: right;
  width: 150px;
  height: 100px;
  background-color: lightblue;
}

.clear-fix {
  clear: both;
  background-color: gray;
  padding: 10px;
}
